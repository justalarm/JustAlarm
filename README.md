# Just Alarm

Just Alarm은 Windows용 간단한 알람 앱입니다. 

알람 기능 외에는 아무것도 없습니다.

말 그대로 Just Alarm입니다.

<img src="https://justalarm.github.io/assets/images/screenshot.png" width="50%">


## 다운로드

[Windows 11 x64](https://github.com/justalarm/JustAlarm/releases/JustAlarm-Setup.exe)를 설치파일을 받아 실행하세요.

전체 릴리즈 [Github Release](https://github.com/justalarm/JustAlarm/releases/tag/v1.3)에서 확인할 수 있습니다.

## 빌드, 개발하기

개발 환경 준비, 빌드 명령, 프로젝트 구조, 설치 파일 제작은 [개발 문서](./DEVELOPMENT.md)에 정리해 두었습니다.
